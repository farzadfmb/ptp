🔍 راهنمای حل مشکل صفحه خالی

صفحه دوره‌ها خالی است؟ این راهنما کمک می‌کند!

✅ دوره‌های نمونه ایجاد شد

4 دوره نمونه در دیتابیس ایجاد شده است:

  • دوره جامع (قبلاً موجود بود)
  • دوره مهارت‌های ارتباطی
  • مدیریت زمان و بهره‌وری
  • رهبری و مدیریت تیم

⚠️ احتمالاً مشکل از Session است

دلایل محتمل خالی بودن صفحه:

  • وارد سیستم نشده‌اید: باید ابتدا وارد پنل سازمان شوید
  • Organization ID متفاوت است: دوره‌ها برای organization_id = 1 ایجاد شده‌اند
  • دسترسی ندارید: باید دسترسی courses_view یا courses_manage داشته باشید

1
بررسی وضعیت

ابتدا وضعیت session و دیتابیس را بررسی کنید:

🔍 صفحه Debug

2

اگر وارد نشده‌اید، ابتدا وارد پنل سازمان شوید:

http://localhost:8888/ptp/organizations/login

نکته: مطمئن شوید که organization_id کاربر شما 1 است.

3
مشاهده صفحه دوره‌ها

پس از ورود، به صفحه دوره‌ها بروید:

📚 صفحه دوره‌ها

4
اگر باز هم خالی است

راه‌حل‌های احتمالی:

  • کنسول مرورگر (F12) را باز کنید و خطاهای JavaScript را بررسی کنید
  • Cache مرورگر را پاک کنید (Ctrl+Shift+Delete)
  • صفحه را Refresh کنید (Ctrl+R یا F5)
  • از مرورگر دیگری امتحان کنید

5
ایجاد دوره جدید
