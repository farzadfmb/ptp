🧪 تست عناصر صفحه درس‌ها

این صفحه برای بررسی اینکه آیا عناصر به درستی نمایش داده می‌شوند یا نه

1️⃣ تست دکمه افزودن درس

لیست درس‌ها

✓ اگر دکمه بالا را می‌بینید، یعنی دکمه به درستی کار می‌کند

2️⃣ تست حالت خالی (وقتی درسی نیست)

هنوز هیچ درسی اضافه نشده است

برای شروع، اولین درس دوره را اضافه کنید

✓ اگر پیام بالا را می‌بینید، یعنی empty state کار می‌کند

3️⃣ بررسی مشکل احتمالی

  • آیا وارد سیستم شده‌اید؟ باید ابتدا login کنید
  • آیا URL صحیح است؟ باید شامل course_id باشد
  • آیا خطایی در Console وجود دارد؟ F12 را بزنید و Console را چک کنید
  • آیا CSS لود شده است؟ Network tab را بررسی کنید

4️⃣ لینک‌های مفید

🔗 صفحه دوره‌ها

🔗 ورود به سیستم

⚠️ مراحل عیب‌یابی

  1. مطمئن شوید که وارد سیستم شده‌اید
  2. به صفحه دوره‌ها بروید
  3. روی دکمه سبز "درس‌ها" کنار یک دوره کلیک کنید
  4. اگر صفحه خالی بود، کلید F12 را بزنید
  5. در تب Console خطاها را بررسی کنید
  6. در تب Network ببینید آیا فایل‌های CSS و JS لود شده‌اند
  7. Screenshot از خطا یا صفحه خالی برای من بفرستید
